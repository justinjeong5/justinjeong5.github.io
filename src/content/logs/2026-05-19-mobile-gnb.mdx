## 문제

iPhone 15(393×852)에서 GNB가 viewport를 넘어 가로 스크롤. 계산:

- brand 42 + padding 32 + nav links 5개 × 평균 60 + actions 3개 × 38 + gap = **484px**
- viewport 393px에서 91px 초과

## 한 일

- 720px 이하 미디어 쿼리: `.primary-nav { display: none }` + 햄버거 트리거 표시
- `components/ui/MobileMenu.jsx` 신설
  - 우측 슬라이드인 240ms cubic-bezier
  - Esc·바깥 클릭·라우트 변경 시 자동 닫힘
  - 메뉴 안에 PRIMARY (Cases·Notes·Essays·Logs·About) + SECONDARY (Uses·Now·Reading·CV) 모두 노출
- 키보드 도움말 버튼은 `.desktop-only`로 모바일 숨김 (모바일에 키보드 없음 = 무의미)
- UIProvider에 `menuOpen`·`openMenu`·`closeMenu`·`toggleMenu` 추가

## 새 모바일 GNB 구조

```
[JKH] ........... [🔍][☀/🌙][≡]
  42px         search theme menu
```

폭 약 220px. 393px viewport에 여유 170px.

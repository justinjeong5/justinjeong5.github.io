## 📚 기술 스택
- React
  - Component간에 Prop로 데이터 교환
  - Redux를 이용한 상태관리
  - Redux-Saga의 비동기 통신과 상태관리
  - React-Router-Dom등의 라이브러리 사용
  - HOC 사용
  - Ant Design을 이용한 디자인
- NextJs
  - GetServerSideProps를 이용한 SSR
  - Head, Link 등의 Built-in 컴포넌트 사용
  - Dynamic Routing을 이용한 페이지 설정
